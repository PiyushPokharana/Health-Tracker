## Filtering Notes

### Open Filter Panel

1. **Tap Filter Icon**
   - Look for the filter icon in the top-right
   - Filter panel appears below search bar

### Filter by Status

**Toggle Status Chips:**
- âœ… **Complete** - Green checkmark
- âŒ **Missed** - Red X
- â– **Skipped** - Yellow dash

**How it Works:**
- Selected = Show those notes
- Unselected = Hide those notes
- Can select any combination

**Example:**
- Want only completed days? Toggle OFF missed and skipped

### Filter by Habit (All Notes View Only)

**When Viewing All Notes:**
1. Scroll to "Habits" section in filter panel
2. See chips for each of your habits
3. Toggle habits ON/OFF
4. Only selected habits show notes

**Example:**
- Have "Exercise", "Reading", "Meditation"
- Toggle OFF "Meditation"
- See only Exercise and Reading notes

### Filter by Date Range

1. **Tap "Select Date Range"**
   - Button in filter panel

2. **Pick Dates**
   - Start date and end date
   - Use the date range picker

3. **See Results**
   - Only notes within that range appear

4. **Clear Range**
   - Tap "Clear" next to date range label

**Example:**
- Pick Nov 1 - Nov 7
- See only notes from that week

### Combining Filters

**Multiple Filters Work Together:**

Example: "Show me ONLY complete status from Exercise habit in November"
1. Search: "Exercise" or use habit filter
2. Status: Toggle ON only âœ… Complete
3. Date: Nov 1 - Nov 30

## Working with Notes

### Edit an Existing Note

1. **From Calendar**
   - Open habit detail screen
   - Tap the day with a note (has âšª indicator)
   - Edit note in bottom sheet
   - Tap "Save"

2. **From Notes List**
   - Tap on any note card
   - Opens habit detail screen
   - Calendar shown for that habit
   - Tap the day to edit

### Delete a Note

1. **Open the Day**
   - Tap the day in calendar

2. **Delete Note**
   - Tap "Delete" button
   - Confirm deletion
   - Note and record are removed

### Change Status (Keep Note)

1. **Open the Day**
   - Tap the day with a note

2. **Change Status**
   - Tap a different status button
   - Note is preserved
   - Tap "Save"

## Troubleshooting

### "I can't find a note I just added"
**Check:**
- Is the status filter including that status?
- Is the habit selected in habit filter?
- Is the date within the date range?

### "Notes not showing on calendar"
**Possible Causes:**
- Note might be empty (blank notes don't show indicator)
- Check the correct habit
- Ensure note was saved (tap Save button)

### "Search not working"
**Try:**
- Clear search and try again
- Check spelling
- Try searching habit name instead
- Make sure filters aren't excluding results
